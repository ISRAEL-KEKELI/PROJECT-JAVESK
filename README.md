# Javesk Learning Platform

Javesk is a mobile-first learning platform tailored for Ghanaian youth, focusing on practical, high-demand skills aligned with emerging industries (tech, renewable energy, agribusiness, creative economy).

## Features
- Micro-courses with localized content
- AI-driven personalized learning paths
- Mentorship and job placement modules
- Certificates and gamification
- Offline mode for limited connectivity

## Tech Stack
- React Native (Expo)
- Firebase / Supabase
- Paystack / Flutterwave
- Tailwind RN (optional)
